# Dower Coppler Paper

Standalone paper repository for the Dower Coppler manuscript. This repo is intentionally separate from `~/dev/caterpillar`; the parent caterpillar repository is not used for commits here.

## Reproducing figures

The derived NPZ inputs used by the paper are committed under `data/`.
Regenerate the simulation table, figures, and PDF with:

```bash
python3 scripts/phase_ls_research_sweep.py
python3 scripts/generate_paper_figures.py
latexmk -pdf -interaction=nonstopmode paper.tex
```

The generated figures are written to `outputs/paper_figures/`. The current compiled manuscript is `paper.pdf`.
The generated simulation table is `outputs/paper_stats/simulation_results_table.tex`;
the full machine-readable sweep is in `outputs/research_sweep/`.
Figure 6 uses the compact temporal-window source
`data/head_2025-09-21_temporal_windows_plane4.npz`, so it can be rebuilt
without the large per-acquisition sidecar.

The "different recording" figure is configurable without editing the paper:

```bash
python3 scripts/generate_paper_figures.py \
  --external-recording-data data/bt24480388_2026-05-18_152605_txel0_h5_row-1_fine_xz_y-3p5to3p5mm_10elev_all20.npz \
  --external-recording-plane -1
```

`--external-recording-plane -1` selects the middle elevation plane.

To refresh the compact temporal-window source from the full per-acquisition
sidecar, run:

```bash
python3 scripts/generate_paper_figures.py --refresh-temporal-summary \
  --temporal-per-acq-dir /Users/lev/dev/caterpillar/results/doppler_cnr_gui/head_2025-09-21_full2dtx_fast8_fine_xz_y-4to4mm_10elev_acq000_479_per_acq
```

## Provenance

See `reproducibility/manifest.json` for hashes, source ultratrace paths, acquisition ranges, and notes for each NPZ input.
The manifest intentionally tracks deterministic figure PNGs but not the
Matplotlib-generated figure PDFs, whose embedded PDF metadata can change on
regeneration even when the visual content is unchanged. The manuscript PDF
itself remains hash-tracked.

For the primary September 21 source ultratrace
`/mnt/pocampus/lev/ultratrace_Head_monster_2025-09-21_21-32-01_y-20to20mm_30elev.h5`,
the HDF5 acquisition config stores `num_angles=5` and
`num_loops=700`, with raw `iq_frames` shaped
`(702, 5, 8, 134, 88)` for acquisition 200. The two extra loops are
noise loops. The runtime metadata reports an empirical transmit pulse
PRF of 1244.224 Hz for acquisition 200. The all-480 fine-grid paper
outputs store an empirical full-ultratrace pulse PRF of 1240.909 Hz,
so the compounded slow-time cadence used for phase-to-velocity
conversion is PRF / 5, approximately 248.2 Hz.

The May 18 external-recording NPZ uses source ultratrace
`/mnt/pocampus/lev/may18_txel_sweep_minus5_0_plus5/ultratrace_BT24480388_monster_2026-05-18_15:26:05.h5`.
Its H5 PRF is also a transmit pulse PRF with five compounded transmit
angles, so the stored Doppler cadence is PRF / 5: pulse PRF 1476.262 Hz
and compounded-frame cadence 295.252 Hz.

Figure 4 CNR values use the largest tolerant circle centered inside each exported segmented ROI, requiring at least 80% overlap with the exported mask, plus one shared background rectangle. The code path is `load_region_export()` -> `largest_tolerant_circle()` in `scripts/generate_paper_figures.py`. The split-half visual consistency figure uses `data/head_2025-09-21_split_half_plane4_all480.npz`; the split-half ROI sign-agreement statistic is regenerated with `python3 scripts/compute_split_half_sign_agreement.py`.

Table 2 is regenerated by `python3 scripts/phase_ls_research_sweep.py`.
The paper includes the generated LaTeX table block from
`outputs/paper_stats/simulation_results_table.tex`; the summary JSON records
the exact defaults (`T=64`, `N=1024`, `K=5`, `omega={0.35,0.8,1.35}`
rad/frame, and SNR `-12..12` dB).
